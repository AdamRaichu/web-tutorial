Introduction

Modern websites are built on 3 main languages: HyperText Markup Language (HTML); Cascading Style Sheets (CSS); and JavaScript (JS).

HTML

HTML documents contain the basic text and images that make up web pages, as well as links to other resources (such as CSS and JS files).

JS

JavaScript is the most complicated of the 3 languages mainly because it has so many different things to learn. With HTML and CSS, it's simply a matter of learning the basics and picking up new tricks as you continue coding. With JS, there are a massive number of keywords, you need to learn operators, AND every single JS file you run can add more functions to the global context. (That's a fancy way of saying the main JS context.) However, it is the most useful of the 3 for making interesting content such as games.
