Introduction

Modern websites are built on 3 main languages: HyperText Markup Language (HTML); Cascading Style Sheets (CSS); and JavaScript (JS).

HTML

HTML documents contain the basic text and images that make up web pages, as well as links to other resources (such as CSS and JS files).

CSS

CSS documents provide visual instructions to the browser for rendering HTML. You can use a variety of selectors for specifying styling, which are applied in order of specificity, from least specific to most specific.
